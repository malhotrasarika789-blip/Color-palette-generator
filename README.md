Color Palette Generator

A simple and interactive Color Palette Generator built using HTML, CSS, and JavaScript.
Generate beautiful color palettes instantly with different formats and tones.

🚀 Live Demo

 https://color-palette-generator-one-dusky.vercel.app

 Features
 Generate random color palettes
 Supports HEX and RGB formats
 Light and Dark tone selection
 Fast and responsive UI
 One-click palette generation

Tech Stack

CSS3
JavaScript (Vanilla JS)
